# Couch 'N Cures - Online Therapy Website

A classy, elegant React website for Couch 'N Cures, a Bangalore-based online therapy space offering compassionate, affordable, and confidential mental health support.

## Design Philosophy

This website features a **classy, elegant, and calming design** that creates a relaxing experience for visitors. The design emphasizes:

- **Elegant Typography**: Serif fonts (Playfair Display, Lora, Cormorant Garamond) for a sophisticated, trustworthy feel
- **Soft Color Palette**: Warm mustard yellow, cream, sage, and black tones
- **Smooth Animations**: Subtle, refined animations using Framer Motion
- **Generous Whitespace**: Clean, uncluttered layouts that breathe
- **Classy Aesthetics**: Traditional design elements with modern functionality

## Features

- **Elegant Design**: Classy, professional aesthetic that feels calm and trustworthy
- **Responsive Design**: Mobile-first design using Tailwind CSS
- **Smooth Animations**: Framer Motion for refined, elegant transitions
- **Brand Colors**: Warm mustard yellow, cream, sage, and black
- **WhatsApp Integration**: Direct booking via WhatsApp
- **Service Information**: Individual Therapy, Couples Therapy, Mental Health Packages, and Well-being Check-ins
- **Testimonials**: Elegant client feedback presentation
- **FAQ Section**: Sophisticated accordion-style questions
- **Contact Information**: Refined contact section with elegant cards
- **Footer**: Privacy Policy, Terms of Use, Copyright, and Social Media links

## Technology Stack

- React 18
- Vite (Build tool)
- Tailwind CSS 3
- Framer Motion (smooth animations)
- JavaScript (JSX)

## Installation

1. Install dependencies:
```bash
sudo npm install
```

2. Start the development server:
```bash
sudo npm run dev
```

3. Build for production:
```bash
sudo npm run build
```

## Customization

### Colors

Brand colors are defined in `tailwind.config.js`:
- `mustard-yellow`: #FFD700 (Primary accent)
- `mustard-warm`: #E6B800 (Hover states)
- `mustard-light`: #FFF176 (Light variant)

### Content

Update the content in the respective component files:
- Services: `src/components/Services.jsx`
- Testimonials: `src/components/Testimonials.jsx`
- FAQ: `src/components/FAQ.jsx`
